# 02 — Математика НМТ: 140 → 175+ за 24 місяці

**Метод — Active Recall на кожному кроці.** Прочитати теорію → закрити книгу → розв'язати на папері. Без підкреслювань, без «перечитати без розв'язання».

**Ресурси (повторюються в усьому плані):**
- mathema.me — основні розділи теорії з вправами
- znoclub.com — фільтрація задач за темою
- osvita.ua — офіційні НМТ минулих років
- Капіносов (підручник) — для важких глав

## ФАЗА 1 — Літо 2026 (1 червня — 31 серпня, 13 тижнів)

- Графік: **45 хв/день, 6 днів/тиждень**
- Ціль на 31 серпня: **150+** на повному mock-тесті.

### Тижні 1–2 (1–14 червня) — Дроби та відсотки

- Додавання / віднімання / множення / ділення дробів, мішані числа, дроби ↔ десяткові, відсоток від числа, знаходження цілого.
- mathema.me → розділ «Дроби» (всі вправи по порядку).
- Щодня: 20 задач з znoclub.com → фільтр «Дроби та відсотки».
- **Milestone:** 20 правильних поспіль без калькулятора.

### Тижні 3–4 (15–28 червня) — Степені та корені

- Закони показників: aᵐ·aⁿ = aᵐ⁺ⁿ, aᵐ/aⁿ = aᵐ⁻ⁿ, (aᵐ)ⁿ = aᵐⁿ.
- Від'ємні показники, a⁰ = 1, корені, √(a²b) = a√b.
- mathema.me → «Степені та корені».
- Щодня: 15 задач. **Drill:** √(a²+b²) ≠ a+b — це найчастіша помилка, розв'язати 20 прикладів саме цього типу.
- **Milestone:** будь-який кореневий вираз спрощений за <90 секунд.

### Тижні 5–6 (29 червня — 12 липня) — Формули скороченого множення

- (a±b)² = a² ± 2ab + b², a² − b² = (a+b)(a−b), (a±b)³.
- Розкладання на множники + групування.
- mathema.me → «Формули скороченого множення».
- Щодня: 15 задач — мікс «розкрити дужки» + «розкласти на множники».
- **Milestone:** розкласти на множники будь-який многочлен 2-го степеня за <2 хв.

### Тижні 7–8 (13–26 липня) — Лінійні рівняння та системи

- ax + b = 0, текстова → рівняння, графічний сенс розв'язку.
- Системи 2×2 (підстановка + додавання), текстові → системи.
- mathema.me → «Рівняння. Системи».
- Щодня: 15 задач. **50% часу на текстові задачі** — формат НМТ.
- **Milestone:** переклад будь-якої текстової задачі в систему за <3 хв.

### Тижні 9–10 (27 липня — 9 серпня) — Квадратні рівняння

- ax² + bx + c = 0, дискримінант D = b² − 4ac, випадки D > 0 / = 0 / < 0.
- Теорема Вієта: x₁ + x₂ = −b/a, x₁·x₂ = c/a.
- Текстові квадратні задачі.
- mathema.me → «Квадратні рівняння» + глава Капіносова.
- Щодня: 20 задач.
- **Milestone:** будь-яке квадратне за <90 сек через дискримінант.

### Тижні 11–12 (10–23 серпня) — Базова планіметрія

- Теорема Піфагора + обернена, подібність трикутників (AA, SAS, SSS), рівність трикутників.
- Площі: трикутник ½bh, паралелограм bh, трапеція ½(a+b)h, круг πr².
- Периметри, медіана / висота / бісектриса.
- mathema.me → «Геометрія» → планіметрія базова.
- Щодня: 15 задач. **Правило: малювати схему для КОЖНОЇ задачі, завжди.**
- **Milestone:** базова задача на трикутник / площу без формульного листа.

### Тиждень 13 (24–31 серпня) — Перший повний mock + reset

- День 1: офіційний НМТ 2024 з osvita.ua. Рівно **150 хв, без телефону**.
- Дні 2–3: категоризувати кожну помилку — *неуважність / обчислювальна / невідома формула*.
- Дні 4–7: drill **тільки топ-1 категорії помилок**, 30 задач/день.
- **Ціль: 150+.** Якщо < 145 — повторити 2 найслабші теми перед стартом Фази 2.

## ФАЗА 2 — 10-й клас (вересень 2026 — травень 2027, 36 шкільних тижнів)

- Графік: **30 хв у будні, 60 хв у вихідні**.
- Метод: шкільна програма + НМТ-практика по тій же темі **відразу**.
- Кожна сесія: поточна тема (20 хв) + **2–3 задачі з Фази 1** (10 хв) = interleaving.
- Ціль на травень 2027: **165+** на повному mock.

### Вересень 2026 (тижні 1–4) — Квадратні нерівності

- ax² + bx + c > 0, метод парабол (відкрита вгору/вниз), системи нерівностей, метод інтервалів.
- mathema.me → «Нерівності» → квадратні + Капіносов.
- Щодня по НМТ: 10 задач з znoclub.com → «Нерівності».
- Interleaving: кожну сесію закінчувати 2 квадратними рівняннями з Фази 1.

### Жовтень 2026 (тижні 5–8) — Функції: core

- Означення функції, ОДЗ, область значень, парність/непарність.
- Лінійна: kутовий коефіцієнт k, b, паралельність k₁ = k₂, перпендикулярність k₁·k₂ = −1.
- Квадратична: вершина xᵥ = −b/2a, напрямок гілок за знаком a, нулі, читання графіка.
- Зростання/спадання з графіка.
- mathema.me → «Функції» (всі підрозділи) + **Desmos.com** — кожну функцію вбити в Desmos і звірити з ручним ескізом.
- **Фокус НМТ:** ОДЗ зустрічається щороку. 20 ОДЗ-задач/тиждень.
- Щодня: 10 задач. **Завжди малювати ескіз перед розв'язанням.**

### Листопад 2026 (тижні 9–12) — Показникова і логарифмічна функції

- y = aˣ (a > 1 росте, 0 < a < 1 спадає), точка (0,1) завжди на графіку.
- y = logₐx (ОДЗ: x > 0 строго), log₂8 = 3 бо 2³ = 8.
- Натуральний log = lnₑ.
- Властивості: log(ab) = log a + log b, log(a/b) = log a − log b, log(aⁿ) = n·log a.
- Перехід основи: logₐb = log b / log a.
- mathema.me → «Показникова функція» + «Логарифмічна функція».
- **Фокус НМТ:** ОДЗ для log (аргумент > 0, основа > 0 і ≠ 1) — головне джерело помилок. 15 ОДЗ-задач для log/тиждень.
- Щодня: 10 задач.

### Грудень 2026 (тижні 13–16) — Тригонометрія, частина 1

- Одиничне коло (малювати від руки, поки не заб'ється в пам'ять).
- Точні значення sin / cos / tan для 0°, 30°, 45°, 60°, 90°, 120°, 135°, 150°, 180°, 270°, 360°.
- Радіани: π = 180°, π/6 = 30°, π/4 = 45°, π/3 = 60°, π/2 = 90°.
- Основна тотожність: sin²α + cos²α = 1, tan α = sin α / cos α.
- mathema.me → «Тригонометрія» → тригонометричні функції.
- **Метод:** малювати одиничне коло з нуля щодня протягом 2 тижнів. Без формульного листа. Записувати sin/cos/tan для всіх 12 ключових кутів з пам'яті.
- Щодня: 10 задач. 50% — читання з кола, 50% — застосування тотожності.

### Січень 2027 (тижні 17–20) — Тригонометрія, частина 2

- Формули зведення: sin(π−α) = sin α, sin(π+α) = −sin α, cos(π/2−α) = sin α тощо.
- Подвійний кут: sin 2α = 2 sin α cos α, cos 2α = cos²α − sin²α = 1 − 2 sin²α = 2 cos²α − 1.
- Базові тригонометричні рівняння: sin x = a (|a| ≤ 1), cos x = a, tan x = a — загальні формули.
- Розв'язання на [0; 2π].
- Капіносов → тригонометрія + mathema.me → «Тригонометричні рівняння».
- **НМТ:** зазвичай 2–3 тригонометричних, найчастіший формат — рівняння.
- Щодня: 10 задач.

### Лютий 2027 (тижні 21–24) — Координатна геометрія і вектори

- Відстань √((x₂−x₁)² + (y₂−y₁)²), середина відрізка.
- Рівняння прямої: y = kx + b і ax + by + c = 0, знаходження k за двома точками.
- Відстань від точки (x₀, y₀) до прямої: d = |ax₀ + by₀ + c| / √(a² + b²).
- Вектори: координати, додавання, множення на число, |a⃗| = √(x² + y²).
- Скалярний добуток: a⃗·b⃗ = x₁x₂ + y₁y₂ = |a||b|cos θ.
- Умови колінеарності / перпендикулярності.
- mathema.me → «Координати та вектори».
- Щодня: 10 задач — мікс «координати + вектор» у кожній сесії.

### Березень 2027 (тижні 25–28) — Похідна

- Похідна як миттєва швидкість зміни (без формальних меж).
- Правила: (C)' = 0, (xⁿ)' = nxⁿ⁻¹, (sin x)' = cos x, (cos x)' = −sin x, (eˣ)' = eˣ, (ln x)' = 1/x, (aˣ)' = aˣ·ln a.
- Сума/різниця, добуток (uv)' = u'v + uv', ланцюгова (f(g(x)))' = f'(g)·g'.
- Екстремуми: f'(x) = 0 → перевірка зміни знаку. Зростання: f'(x) > 0. Спадання: f'(x) < 0.
- Дотична: y − y₀ = f'(x₀)(x − x₀).
- mathema.me → «Похідна та її застосування» + Khan Academy «Derivatives intro» (англ., B1 достатньо для відео).
- **НМТ:** передбачувані формати — знайти екстремум, кутовий коефіцієнт дотичної, інтервал зростання.
- Щодня: 10 задач.

### Квітень 2027 (тижні 29–32) — Просунута планіметрія

- Вписаний кут = ½ центрального.
- Дотична ⊥ радіус у точці дотику. Дві дотичні з однієї зовнішньої точки рівні.
- Кут між хордами, перетин січних і дотичних.
- Відношення площ подібних = (коефіцієнт подібності)².
- Медіана, бісектриса, висота — довжини в трикутнику.
- Радіуси вписаного і описаного кола.
- Капіносов → геометрія просунута + mathema.me → планіметрія просунута.
- Щодня: 15 задач. Малювати схему. Підписувати все, що дано.

### Травень 2027 (тижні 33–36) — Серія mock-тестів

- Повний mock кожні 2 тижні (всього 2 у травні).
- Тести: офіційні НМТ 2022, 2023 з osvita.ua.
- Після кожного: список помилок за темою + категорією (неуважність / обчислення / невідома формула).
- Якщо одна тема в логу помилок 3+ разів → гап, 5 днів на неї.
- Трекати delta від тижня до тижня.
- **Ціль: 165+ до 31 травня.** Якщо < 160 — перші 2 тижні літа дати топ-2 темам помилок.

## ФАЗА 3 — Літо 2027 (1 червня — 31 серпня, 13 тижнів)

- Графік: **60 хв/день, 6 днів/тиждень**.
- Ціль 31 серпня: **170+**. Якщо не досягнуто — продовжити симуляцію у вересень.

### Червень 2027 (тижні 1–4) — Логарифмічні і показникові рівняння

- Показникові: метод рівних основ (2ˣ = 8 → x = 3), різні основи (2ˣ = 3 → x = log₂3 = ln 3 / ln 2).
- Метод заміни (t = 2ˣ, квадратне в t).
- Логарифмічні: logₐx = b → x = aᵇ; logₐf(x) = logₐg(x) → f(x) = g(x) з перевіркою ОДЗ.
- Комбіновані, заміна.
- mathema.me → «Логарифмічні та показникові рівняння» + Капіносов.
- Щодня: 20 задач. **Завжди перевіряти ОДЗ у логарифмічних — більшість екзаменаційних помилок саме тут.**

### Липень 2027 (тижні 5–8) — Стереометрія

- Прямокутний паралелепіпед: V = abc, діагональ = √(a² + b² + c²).
- Призма: V = S_осн × h, бічна поверхня = периметр × h.
- Піраміда: V = ⅓ × S_осн × h, апофема через Піфагора.
- Циліндр: V = πr²h, повна поверхня = 2πr² + 2πrh.
- Конус: V = ⅓πr²h, твірна l = √(r² + h²), повна поверхня = πrl + πr².
- Куля: V = ⁴⁄₃πr³, поверхня = 4πr².
- Перетини: діагоналі перерізу, кут між гранню і основою.
- mathema.me → «Стереометрія» + Капіносов.
- **НМТ:** 1–2 задачі/тест, майже завжди об'єм або поверхня. **5 формул вивчити досконало.**
- Щодня: 15 задач. Малювати 3D-фігуру і підписувати всі розміри.

### Серпень 2027 (тижні 9–13) — Режим повної симуляції

- **Тиждень 9:** повний mock (150 хв, без телефону, без перерв) → лог помилок.
- **Тиждень 10:** drill топ-2 теми помилок (20 задач/день/тема).
- **Тиждень 11:** повний mock → delta до Тижня 9.
- **Тиждень 12:** drill топ-2 з Тижня 11.
- **Тиждень 13:** фінальний mock → ціль **170+**.
- Ресурси: офіційні НМТ 2021, 2022, 2025 + znoclub.com за слабкими темами.

## ФАЗА 4 — 11-й клас (вересень 2027 — травень 2028, 36 тижнів)

- Графік: **45 хв у будні, 90 хв у неділю (mock-день)**.
- **Кожна неділя — повний mock, без винятків. Кожна середа — розбір помилок + 15-задачний drill.**
- Ціль: **3 поспіль mock'и на 170+** = готовність. Фінальна ціль **175+**.

### Вересень — жовтень 2027 (тижні 1–8) — Інтеграли

- Первісна F(x) така що F'(x) = f(x).
- Таблиця: ∫xⁿdx = xⁿ⁺¹/(n+1) + C (n ≠ −1), ∫sin x dx = −cos x + C, ∫cos x dx = sin x + C, ∫eˣdx = eˣ + C, ∫1/x dx = ln|x| + C.
- Визначений інтеграл ∫ₐᵇf(x)dx = F(b) − F(a).
- Геометричний сенс — площа між кривою і віссю x.
- Площа між двома кривими: ∫ₐᵇ|f(x) − g(x)|dx.
- mathema.me → «Інтеграл» + Khan Academy «Integrals».
- **НМТ:** 1 задача, формат завжди той самий — обчислити визначений інтеграл або площу.
- Щодня: 10 задач.

### Листопад — грудень 2027 (тижні 9–16) — Комбінаторика і ймовірність

- Основний принцип лічби, перестановки P(n) = n!, розміщення A(n,k) = n!/(n−k)!, сполучення C(n,k) = n!/(k!(n−k)!).
- Трикутник Паскаля, біном Ньютона базово.
- Класична ймовірність P = m/n.
- Доповнення P(Ā) = 1 − P(A).
- Незалежні події P(A ∩ B) = P(A) · P(B).
- Геометрична ймовірність.
- mathema.me → «Комбінаторика та ймовірність».
- **НМТ:** 2–3 задачі. **Найлегша зона для бонусних балів — лише застосувати правильну формулу.**
- Щодня: 10 задач. Особлива увага сполученням — найчастіше зустрічається.

### Січень — лютий 2028 (тижні 17–24) — Задачі з параметрами

- Рівняння з параметром a: знайти всі a, для яких рівняння має розв'язки / рівно один / не має.
- Параметричні нерівності, графічний метод (сім'я прямих, умова через дискримінант).
- Капіносов → «Рівняння та нерівності з параметром» (найскладніша глава) + znoclub.com → «Задачі з параметром».
- **НМТ:** 1 складна задача, значний внесок у бал. **Стратегія: завжди намагатися, писати всі міркування — можлива часткова оцінка.**
- Щодня: 5–8 задач (матеріал повільний, не поспішати).
- **Тригер репетитора:** якщо за 2 тижні нуль прогресу на параметричних — найняти на 4 точкові сесії саме по цій темі (~20–40 €).

### Березень — травень 2028 (тижні 25–36) — Симуляція іспиту, без нового матеріалу

- **Кожна неділя:** повний офіційний mock (150 хв, точні умови іспиту).
- **Кожна середа:** аналіз помилок неділі + 15-задачний drill на найслабшу тему.
- Записувати кожну недільну оцінку в лог + обчислювати ковзне середнє за 4 тижні.
- **Останні 2 тижні перед НМТ:** лише офіційні минулі тести, нічого нового.
- Готовність = 3 неділі поспіль 170+.
- Якщо < 165 у квітні — онлайн-репетитор 2 сесії/тиждень тільки на гапи.

## Загальний тригер репетитора

- Якщо оцінка mock'у не росте на 5+ балів **за 4 тижні поспіль** → найняти репетитора.
- Ставка: 5–10 €/год. Платформи: Preply або українські репетитори в Telegram-групах для НМТ.
- Максимум 1–2 сесії/тиждень, **тільки під конкретну тему**, не «загальна підготовка».
