# SK-2028 — план вступу до TUKE FEI Košice (Кібербезпека)

24-місячний план самопідготовки для безкоштовного вступу до **Технічного університету в Кошице, факультет електротехніки та інформатики (FEI), спеціальність «Kybernetická bezpečnosť»**. Старт навчання — вересень 2028.

> **Трек:** лише Словаччина. Польща виключена, бо сертифікат **PKPZJP C1** видається лише з 18 років, а на момент вступу 2028 студент ще буде неповнолітнім.

## Хто це і навіщо

- 16-річний(-я) учень(-иця) з Вознесенська (Миколаївська обл., активна зона бойових дій).
- Школа: 10-й клас з вересня 2026, 11-й клас з вересня 2027, випуск і НМТ — весна-літо 2028.
- Мета: безкоштовне навчання у держ. ВНЗ Словаччини (програма викладається словацькою → tuition безкоштовний для всіх, у т.ч. громадян України).

## Ключові цифри (одна сторінка)

- **НМТ-математика:** 140 зараз → ціль **175+** до травня 2028 (24 міс підготовки).
- **Англійська:** B1 → B2 до літа 2027 (передумова для SAT, якщо знадобиться).
- **Словацька:** A0 → B1 сертифікат (ECL або ŠJS на FF UK) до весни 2028.
- **Бюджет 1-го року в Кошице:** ~6 600 €. Загалом до літа 2028 потрібно **~10 000 €**.
- **Реалістичні накопичення:** ~7 750 € → **гап ~2 250 €** (закривається через SAIA / стипендії MŠVVaŠ SR / робота на кампусі від 1-го курсу).
- **VNJ (заморожена сума на рахунку для police):** ~3 400 € — НЕ витрачати, повинно існувати на момент подачі.

## Принцип роботи з планом

- **Floor-звичка щодня:** 10 хв Anki + 1 задача з математики. Це **мінімум**. Не більше нуля.
- **Тиждень = успішний, якщо 5 днів із 7** виконано. Не binary streak.
- **Active recall** — єдиний метод вивчення: прочитав теорію → закрив книгу → розв'язав на папері. Без підкреслювань і «перечитати».
- **Не пропускати двічі поспіль.** Один пропуск — людина. Два — новий патерн.
- **Identity frame:** «я людина, яка щодня робить крок до переїзду».

## Чого тут НЕМАЄ і чому

- **Python / програмування до вступу** — спочатку математика і словацька. Python почнеться у TUKE з 1-го курсу.
- **Dual-track PL + SK** — Польща заблокована (див. вище).
- **Pomodoro-додатки, ICS-експорт, селектори хронотипу** — зайва складність, не методи навчання.
- **90 хв/день мат на старті** — гарантоване вигорання до 6-го тижня. Стартуємо з 45 хв.
- **«Заморозити чужі гроші для виписки»** — police може вимагати виписку за 3 місяці. Це ризик відмови.
